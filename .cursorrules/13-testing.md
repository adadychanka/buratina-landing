# Testing and Development

## Manual Testing

- Test forms thoroughly before deployment
- Verify all translations work correctly
- Test responsive design on multiple devices
- Check accessibility (a11y) requirements
- Verify API routes work with proper error handling

## Browser Testing

- Test in Chrome, Firefox, Safari, Edge
- Test on mobile devices (iOS, Android)
- Check cross-browser compatibility
- Verify responsive breakpoints

## Accessibility

- Use semantic HTML
- Provide alt text for images
- Ensure keyboard navigation works
- Check color contrast
- Test with screen readers (future)

## Performance Testing

- Use Lighthouse for performance audits
- Check Core Web Vitals
- Optimize images and assets
- Minimize bundle size

## Translation Testing

- Verify all locales load correctly
- Check language switcher works
- Ensure all text is translated
- Test locale detection
